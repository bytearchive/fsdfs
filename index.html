fsdfs¶

Installation¶

The easiest way to get fsdfs is if you have setuptools installed:

easy_install fsdfs

Without setuptools, it’s still pretty easy. Download the fsdfs.tgz file from fsdfs’s Cheeseshop page, untar it and run:

python setup.py install
